# Python Traceroute Utility

## Overview
A Python implementation of a basic traceroute tool using the "scapy" library.

## Requirements
- Python 3.x
- Scapy libary ('pip install scapy')

## Usage

'''bash
python tracert.py google.com -m 20 -t 3
